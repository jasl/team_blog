# TeamBlog

A simple engine to build a team blog

## Demo
go to ```test/dummy``` and ```rails s```, remember ```bundle``` first.

## Installation
```mount TeamBlog::Engine => '/blog'``` in your ```config/routes.rb```.

## Configuration
See ```test/dummy/config/initializers/team_blog.rb```.
